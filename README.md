# Meeow - Desktop Cat Companion 
 
A pixel cat that lives on your Windows desktop. 
 
## Features 
- Eyes follow your cursor 
- Paws react to typing 
- Hearts on scratch 
- Sleep mode with zzz 
- Stretch reminder every 30 min 
- Pomodoro timer 
- Right-click menu with color picker 
- Auto-launches on Windows startup 
 
## Setup 
pip install pynput 
python cat.py
